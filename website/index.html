Die Maison

Gegründet von einer Frau, die genau wusste, was fehlt

HIMA entsteht im Rheinland — aus der Überzeugung, dass modest fashion keine Nische ist, sondern eine eigene Form von Souveränität. Jede Qualität wird von Hand geprüft, bevor sie unseren Namen trägt.
