# HIMA_Design — Design- & Kreativ-Ordner

_Alle Design- und Kreativ-Assets von HIMA an einem Ort. Stand: 28.06.2026._

## Struktur
| Ordner | Inhalt |
|---|---|
| **01_Logos** | Logo als PNG (Espresso & Ivory) + Logo-/Asset-Kit als HTML |
| **02_Instagram** | Instagram-Motive (Launch, Zitat, Produkt) als JPG + Kampagnen-Layout (HTML) |
| **03_Website** | Website-Hero (JPG mit echtem Foto) + Website-Layouts (HTML) |
| **04_TikTok_Facebook** | (noch leer) — Assets werden über das Prompt-System erzeugt (siehe 08) |
| **05_Visitenkarte** | Visitenkarte (PNG) + Layout (HTML) |

| **09_Design-Dateien_HTML** | Alle interaktiven Original-Layouts (Logo, IG, Website, Kampagne) |

## Nutzung
- **Fertige Bilder:** 01, 02, 03, 05 (PNG/JPG) — direkt verwendbar.
- **Layouts bearbeiten:** HTML-Dateien im Browser öffnen (01, 09) → „Drucken → PDF" oder Screenshot.
- **Neues Material erzeugen (Website/IG/TikTok/FB):** `08_Prompts_und_Guidelines/AI_Creative_Prompt_System.md` — perfekte Prompts für Claude Design & Higgsfield, plus Workflow und Marken-/Rechtsregeln.

## Hinweise
- Fonts in den PNG-Rendern: solide Serife (DejaVu); die finale Didone/Cormorant-Version entsteht in Canva.
- Recolor: gut für klar abgegrenzte Stoffe; für pixelgenaue Ergebnisse KI-Recolor (Canva „Magic"/Photoshop) oder echte Farbfotos.
- Material nie als „Seide" auszeichnen (Bambus = Viskose, Medina/Satin = Polyester).
