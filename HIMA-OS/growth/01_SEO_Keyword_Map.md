## 2. Seitenstruktur (SEO-Architektur)
```
Startseite (Cluster B Haupt-Keyword)
├── /hijabs (Kategorie, Cluster A)
│    ├── /hijabs/seide (Hero-Kollektion)
│    ├── /hijabs/everyday
│    └── /produkt/signature-seiden-hijab (Hero-PDP)
├── /kollektionen (saisonal)
├── /ueber-hima (Marke, Story, ḥimā)
└── /journal (Cluster C + D)
     └── einzelne Artikel
```

## 3. On-Page-Regeln je Seite
- Ein Haupt-Keyword + 2–3 Neben-Keywords je Seite.
- Title (≤60 Z.) mit Keyword vorne; Meta-Description (≤155 Z.) mit Nutzen + CTA.
- H1 = Keyword, H2/H3 strukturiert. Alt-Texte für Bilder. Interne Verlinkung Journal → PDP.
- Ladezeit & Mobile zuerst. Strukturierte Daten (Produkt, Bewertung).
